# JDBC Statement CRUD

A Java JDBC project demonstrating basic CRUD (Create, Read, Update, Delete) operations using the JDBC `Statement` interface with MySQL.

## Features

- Create Database
- Create Table
- Insert Records
- Update Records
- Delete Records
- MySQL Database Connectivity
- JDBC Statement API

## Technologies Used

- Java
- JDBC
- MySQL
- Maven
- Eclipse IDE
